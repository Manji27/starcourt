★ À propos du concept

Concept fictif imaginé et intégré
par Marcus Mendy

Le Starcourt Mall n'existe pas, du moins pas dans notre dimension. Cette landing est un projet portfolio inspiré des malls américains des années 80, conçu pour explorer une direction artistique forte et la traduire en code propre, accessible et performant.

01

Direction artistique

Palette pastel années 80, formes Memphis, grain argentique, typographies script + serif éditorial. Tout est dessiné en CSS pur, aucun framework UI.

02

Intégration

HTML sémantique, CSS scopé section par section avec variables namespacées, JS vanilla minimal. Pensé responsive, accessible et léger.

03

Stack

HTML CSS JS vanilla Inter DM Serif Display Pacifico
